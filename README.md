# 🚗 EndlessRiding
### A game where you have to drive through falling rocks and reach your goal of 40 points

## 👨‍💻 Tech Stack
* Java
* Android studio
* Android SDK (Debugger, emulator, ADB)
* Canvas
* Firebase Remote Config
* Firebase Realtime Database
